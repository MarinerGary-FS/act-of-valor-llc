# Act of Valor — Website Build System

## Overview

This repository contains the complete build system for the Act of Valor website.

This is NOT a standard website project.

It is a high-conversion, crisis-response system designed to:
- Capture leads immediately
- Drive phone calls
- Reduce user hesitation during high-stress situations

---

## Business Context

Act of Valor provides:
- Mold Remediation
- Water Damage Cleanup
- Trauma / Biohazard Cleanup
- Fire Damage Restoration

Service Area:
Atlanta Metro Area (including Sandy Springs, Marietta, Douglasville, Buckhead)

Core Differentiators:
- Compassion-first approach
- Trusted by law enforcement and local professionals
- Same-day response

---

## Project Objective

Build a website that:
- Converts within 5–20 seconds
- Prioritizes phone calls above all else
- Feels calm, trustworthy, and responsive
- Performs strongly in local SEO

---

## File Structure

This project uses a structured markdown system to guide AI-assisted development.

### Core Files

- `CLAUDE.md`
  → Master instructions for Claude (behavior, tone, rules)

- `content.md`
  → All website copy (conversion-focused)

- `components.md`
  → UI structure and component definitions

- `design-system.md`
  → Visual identity and styling rules

- `conversion-logic.md`
  → User behavior, CTA placement, and funnel logic

- `seo-strategy.md`
  → Local SEO and ranking strategy

- `interaction-behavior.md`
  → Micro-interactions and user experience behavior
